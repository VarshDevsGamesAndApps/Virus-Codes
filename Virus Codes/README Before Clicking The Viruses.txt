This Is A Document Of Fixes If You Clicked Any Of The Viruses. Please Read Before Executing.

===========================================================================
Harmless Viruses

Settings Virus:
Do Nothing just Keep on pressing the buttons.

===========================================================================

Irritation Computer Virus:
It Is An Loop. So Go To Task Manager, Right Click Windows Script Host, And End Task.

===========================================================================

Matrix Hacked Prank Virus:
Just Exit Command Prompt(CMD).

===========================================================================

Folder Bomber:
Just Exit Command Prompt(CMD).


===========================================================================

Shutdown Prank Virus:
Go To Command Prompt(CMD)And Type 'SHUTDOWN.EXE -A' Immediately.

===========================================================================


Harmful Viruses

All File Fix:
Kill Switch; Shutdown, or Remove Battery And Put it in again, or wait for it fully crash and shutdown.
